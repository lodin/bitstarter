The term deliberate practice is thrown around alot in the tech world. However, no one really goes on to explain exactly how to do it. Read code, talk to other programmers, work on side projects. All very good advice but with so much overhead. Time dedicated to working on a side project is dominated by setup. Meeting programmers will as likely result in beer as hacking and reading code is just plain difficult. Python Rites of Passage is a collection of problems that hits the sweet spot between the triviality toy problems and messiness of real world challenges.

This book is a collection of sizeable programs that are suitable as training projects for the journeyman programmer who wishes to advance his craft. Learning is achieved by reaching slightly higher than your current skill level. No student would expect to read any number of books about painting and be able to create even a good imitation of a Van Gogh. Why, then, expect to read a programming tutorial or two and be able to turn out a well formed program? Each Rite of Passage stands alone with its own background information, problem statement and suggestions for a solution. The solution that follows will only be one of the many possible solutions.

The book will cover 6 problems, and each will ideally take a week or two to work through. Contributers get to vote.Each solution will only be a toy solution. However it should be extensible to a full system.

  2. Templating System
  3. HTML Parser
  4. Lisp Interpreter
  5. Web Framework
  6. Syntax Highlighter
  7. Curl-like utility
  8. Text Editor
  9. Regex Engine
  10. unittest framework
  11. Unix Shell
